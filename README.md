## 📌 개발 배경
저희는 알고리즘에 대한 정치적인 영향을 받고 있습니다. 예를 들어 정치적으로 한 쪽에 편향된 유튜브 영상을 시청한다면 알고리즘이 그 쪽에 관한 영상들만 보여줄 것입니다. 이는 정치적으로 올바른 주관을 가지기 어렵게 하며 점점 반대편의 의견은 보지도 듣지 않게 될 것입니다. 

이에 대한 문제는 언론 매체에서도 쉽게 발견할 수 있습니다. 언론사들은 중립적인 보도를 해야하지만 잘 지켜지지 않고 한 쪽에 유리한 뉴스를 게시합니다.

따라서 하나의 뉴스만 보고 사건을 판단하기 쉽습니다. 이에 대한 문제의 해결책으로 각 사건을 보도한 뉴스를 여러 언론사에서 가져와 AI를 활용하여 사용자는 중립적인 판단을 할 수 있는 앱이 필요하다고 생각하였습니다.

## 🔎 앱 설명
SWEN NEWS는 데이터 사이언스와 AI 기술을 활용하여 한국 언론 생태계의 편향성을 정량적으로 분석하는 차세대 미디어 분석 플랫폼입니다. 

자연어 처리(NLP) 엔진이 실시간으로 한국 주요 언론사의 뉴스 기사를 수집하고 분석하여, 각 매체의 정치적 성향과 편향도를 객관적 수치로 환산합니다. 

딥러닝 기반 감정 분석 시스템이 기사의 감정적 톤과 어조를 실시간으로 평가하여 좌편향, 중도, 우편향 성향을 95% 이상의 정확도로 분류하며, 시계열 분석을 통해 언론사별 편향 트렌드의 변화 패턴을 추적합니다.
